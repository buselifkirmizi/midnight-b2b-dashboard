# midnight-b2b-dashboard
This project aims to build a decentralized B2B dashboard for freelancers and agencies to manage client agreements securely. While the existence of a contract and its timestamp will be public on the ledger, sensitive details like client names, payment amounts, and project scopes will remain completely hidden using Midnight's privacy-first architecture.

Public State vs. Private Witness

Public State: Contract ID, timestamp, and wallet addresses of the involved parties.

Private Witness: The actual agreement details, financial amounts, and client identities. This ensures that business operations can be verified on-chain without leaking competitive or sensitive data.
